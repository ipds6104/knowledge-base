# PIC SPBE KALBAR WhatsApp Chat

## Deskripsi Grup
Grup obrolan WhatsApp ini merupakan grup koordinasi khusus PIC SPBE di lingkungan BPS Provinsi Kalimantan Barat untuk sinkronisasi kebijakan penilai eksternal SPBE tingkat daerah.

## Detail Grup
*   **Kategori**: Koordinasi Statistik Sektoral / SPBE / EPSS Tingkat Provinsi.
*   **Ruang Lingkup**: Penyelarasan timeline penilaian SPBE, instruksi koordinasi provinsi ke kabupaten/kota, dan diseminasi informasi dari BPS RI terkait domain SPBE.
