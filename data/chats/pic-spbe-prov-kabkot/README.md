# PIC SPBE (PROV & KABKOT) WhatsApp Chat

## Deskripsi Grup
Grup obrolan WhatsApp ini digunakan untuk koordinasi penanggung jawab (PIC) Sistem Pemerintahan Berbasis Elektronik (SPBE) Provinsi Kalimantan Barat beserta BPS Kabupaten/Kota se-Kalbar.

## Detail Grup
*   **Kategori**: Koordinasi Statistik Sektoral / SPBE / EPSS.
*   **Ruang Lingkup**: Evaluasi Penyelenggaraan Statistik Sektoral (EPSS), tata cara koordinasi SPBE, pemecahan masalah teknis aplikasi penilai, dan koordinasi administrasi evaluasi tingkat kabupaten/kota.
